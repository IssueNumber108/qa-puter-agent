## üöÄ Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- (Optional) Ollama installed for local models

### Quick Start

1. **Open the application**
   - Simply open `index.html` in your browser

2. **Enter your username and login**
   - Click "Launch QAPuterTalk"
   - Authenticate with Puter

3. **Start chatting!**
   - Select an AI model
   - Type your message
   - Press Enter or click Send

### Local Ollama Setup

1. **Start Ollama with CORS enabled:**
   ```bash
   OLLAMA_ORIGINS=* ollama serve
   ```

2. **Pull a model:**
   ```bash
   ollama pull llama2
   ```

3. **In QAPuterTalk:**
   - Click "Test Connection"
   - Click "Load Available Models"
   - Select your Ollama model
   - Start chatting locally!

For detailed Ollama setup, see [OLLAMA_SETUP_GUIDE.md](OLLAMA_SETUP_GUIDE.md)
